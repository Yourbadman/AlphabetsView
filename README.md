## AlphabetsView
#### Features
* 字母列表支持波浪动画
* 适配多屏幕、RTL模式下显示,类似微信联系人在多屏幕下会自动缩小
* 字母列表支持显示drawable,支持区块字母表选中.
* 弹出的字母表支持显示图片,快速滑动时,交替显示的内容渐变处理，更好的体验
* 字母列表居中显示,支持两两之间配置一个安全padding
